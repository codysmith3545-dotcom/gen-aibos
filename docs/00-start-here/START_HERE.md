# START HERE

GEN-AIBOS is a generic AI Business Operating System foundation.

## What This System Is

A foundation for business data, AI agent actions, workflows, approvals, memory, cost tracking, external system sync, and vertical modules.

## What This System Is Not

It is not a full ERP replacement, QuickBooks replacement, CRM replacement, no-code automation dump, land-only app, or a system where AI gets unrestricted power.

## Golden Rule

Use many tools, but only one canonical business graph.

## First Five Build Steps

1. Create/refine the repo foundation.
2. Read and refine `AGENTS.md`, `CLAUDE.md`, and `system-constitution.md`.
3. Create the Supabase project.
4. Run the core schema migration in `supabase/migrations/0001_core_foundation.sql`.
5. Set up n8n and create the first workflow from `docs/07-workflows/templates/workflow-intake-template.md`.
