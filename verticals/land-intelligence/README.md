# Land Intelligence Module

First planned vertical module for GEN-AIBOS.

## Purpose

Find, enrich, score, and track land opportunities for land wholesaling, entitlement, subdivision, development, and infrastructure-related opportunities.

## Module Rule

Do not modify the generic core to fit land. Extend the core with land-specific tables, workflows, and dashboards.

## Planned Tables

parcels, properties, owners, county_sources, zoning_records, permit_records, utility_records, environmental_constraints, parcel_scores, deal_theses, entitlement_paths, buyer_profiles, due_diligence_items, outreach_campaigns.

## Required Parcel Score Output

score, reason, source evidence, confidence level, missing information, deal-killer risks, human verification needed, next best action, fallback value, estimated entitlement path.
