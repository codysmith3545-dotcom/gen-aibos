# ROI Metrics

## Purpose

GEN-AIBOS should be judged by value, not by tool count. The system should create revenue opportunities, save time, reduce errors, improve decisions, and preserve knowledge.

## Core Rule

A cheaper system is not better if it produces no useful output. A more expensive system can be worthwhile if it creates qualified opportunities or reduces major operational risk.

## Activity Metrics

Track:

- agent_runs_count
- workflow_runs_count
- approvals_created
- approvals_completed
- verification_items_created
- verification_items_closed
- documents_classified
- messages_summarized
- tasks_created

## Value Metrics

Track:

- leads_created
- qualified_opportunities
- proposals_drafted
- projects_advanced
- parcels_screened
- high_value_opportunities_found
- hours_saved_estimate
- mistakes_prevented

## Cost Metrics

Track:

- total_monthly_software_cost
- estimated_agent_cost
- estimated_workflow_cost
- data_collection_cost
- observability_cost
- cost_per_agent_run
- cost_per_workflow_run

## Cost Per Useful Output

Track:

- cost_per_lead_created
- cost_per_qualified_opportunity
- cost_per_parcel_screened
- cost_per_proposal_drafted
- cost_per_hour_saved
- cost_per_mistake_caught

## Continue, Improve, Pause

Continue workflows that save time, create qualified opportunities, improve accuracy, reduce manual review, prevent repeated errors, or support revenue.

Improve workflows that create some value but have quality, cost, or clarity issues.

Pause or retire workflows that create no clear value, repeatedly fail, create bad data, duplicate another workflow, or lack ownership.

## Weekly Review Questions

1. What was the most valuable AI output this week?
2. What was the least valuable automation?
3. Which workflow saved the most time?
4. Which workflow created cleanup?
5. Which output supported revenue?
6. Which cost should be capped?
7. What should be automated next?
8. What should be retired?

## Rule

Do not build dashboards for vanity metrics. Build metrics that help decide what to continue, improve, pause, or scale.
