# GEN-AIBOS — Generic AI Business Operating System

GEN-AIBOS is a modular, governed, AI-operated business foundation where agents can observe, draft, build, test, log, and execute approved workflows against one canonical business graph, while humans retain control over risk, money, legal commitments, client reputation, credentials, and production changes.

## Mission

Create a reusable foundation for AI-operated businesses using GitHub for change control, Supabase/Postgres for the canonical business graph, n8n for integration routing, Google Workspace for communication/files, QuickBooks/Xero for accounting truth, and Claude Code/Codex as implementation/operator agents.

## Core Rule

The foundation stays generic. Business ideas become vertical modules.
