# Core Principles

## Boring by Default

Default AI behavior should be: summarize, draft, recommend, log, and ask approval.

## Build Edge, Buy Commodity

Use existing tools for commodity functions: QuickBooks/Xero, Google Workspace, Stripe, Attio, GitHub, and n8n. Build the canonical graph, approval layer, agent logs, workflow standards, command center, and vertical-specific intelligence.

## One Graph, Many Interfaces

External tools are interfaces. Supabase is the canonical graph.

## Modular by Design

Business ideas become modules, not core rewrites.
